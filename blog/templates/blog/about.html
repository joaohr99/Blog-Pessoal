Minhas Habilidades e Ferramentas

Linguagens e Frameworks
  • Python: Linguagem de programação versátil, base para Data Science e desenvolvimento web.
  • Django: Poderoso framework web para Python, utilizado na construção deste blog.
  • HTML: Linguagem de marcação para estruturar o conteúdo de páginas web.
  • CSS: Linguagem de estilo para dar vida e design visual às páginas web.
  • Bootstrap: Framework CSS que acelera o desenvolvimento responsivo e a estilização de interfaces.
Banco de Dados e Análise de Dados
  • SQL: Linguagem padrão para gerenciar e manipular bancos de dados relacionais.
  • Pandas: Biblioteca Python fundamental para manipulação e análise de dados.
  • Seaborn: Biblioteca Python para visualização estatística de dados, construída sobre Matplotlib.
  • Streamlit: Framework Python para criar rapidamente aplicações web interativas para Data Science.
  • PowerBI: Ferramenta de Business Intelligence da Microsoft para visualização e análise de dados.
Versionamento e Interatividade
  • Git: Sistema de controle de versão distribuído para rastrear mudanças no código.
  • GitHub: Plataforma para hospedagem de repositórios Git, colaboração e portfólio de código.
  • JQuery: Biblioteca JavaScript que simplifica a manipulação de HTML, eventos e animação.
